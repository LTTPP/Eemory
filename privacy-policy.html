Eemory

An Eclipse plug-in aimed at integrating Evernote within the Eclipse IDE

Download .zip Download .tar.gz View on GitHub

Privacy Policy

Information collection and usage

Eemory is a pure client App, and does not collect anything.

Information security

In order to use Eemory you have to authorize it to access your Evernote account, which is done by OAuth. Once authorized, the generated authorization token will be encrypted and stored on your machine as Eclipse metadata. We won't reveal the authorization token to any others.
